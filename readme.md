> CODEBUSKING Helper Extension for Video Guides


## Install

```
$ npm install codebusking-helper
```


## Usage

```js
const codebuskingHelper = require('codebusking-helper');

codebuskingHelper('unicorns');
//=> 'unicorns & rainbows'
```
